# Address Book Management System (C)

A command-line application built in C to manage contact information with data persistence. This project is a deep dive into core C programming and Linux-based software development.

## 🛠 Features
- **CRUD:** Create, Read, Update, and Delete contacts.
- **Persistence:** Save data to the local file system.
- **Memory:** Structured data management using C Structs.

## 💻 Tech Stack
- Language: **C**
- Environment: WSL (Ubuntu) on Windows
- Compiler: GCC (Linux)
